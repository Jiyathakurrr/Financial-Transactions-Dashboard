💳 Financial Transactions Dashboard

An interactive Power BI dashboard analyzing 13M+ financial transactions to uncover customer behavior, revenue trends, transaction methods, and merchant performance.

---

📊 Overview

This project transforms raw transaction-level data into a multi-page executive and analytical dashboard, enabling stakeholders to explore revenue performance, customer segments, and transaction patterns at scale.

---

🧰 Tools & Techniques

| Tools        |                  Purpose                   |
|--------------|--------------------------------------------|
| Power BI     | Dashboard design & visualization           |
| Power Query  | Data cleaning & transformation             |
| DAX          | Custom measures & calculated columns       |
| Data Modeling| Relationship building & star schema design |

🗂️ Dashboard Pages

1️⃣ Executive Dashboard
A high-level summary for quick decision-making.
- KPI Cards
- Revenue by Card Brand
- Revenue by Merchant Category
- Revenue by Gender
- Transaction Type Distribution

2️⃣ Customer Analysis
Deep dive into customer demographics and financial behavior.
- Revenue by Age Group
- Credit Score Analysis
- Income Group Analysis
- Gender Analysis
- Retirement Age Analysis

3️⃣ Transaction Analysis
Granular view of transaction trends and merchant performance.
- Revenue Trend
- Merchant State Analysis
- Merchant City Analysis
- Transaction Method Analysis
- Card Brand vs Card Type

 🚀 Key Insights

• Debit cards account for over 62% of all transactions.
• Mastercard generates the highest total revenue.
• Grocery Stores & Supermarkets contribute the highest merchant revenue.
• Customers aged 45–54 generate the most revenue.
• Low-income customers contribute the largest share of revenue.
